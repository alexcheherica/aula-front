Trabalhando com listas

Listas são estruturas para representação de conteudo na forma de relação de itens.

Listas não-ordenada

Relação de itens em que a sequencia não importa.

Exemplos:

  • Teclado
  • Mouse
  • Webcam
  • Monitor

Lista ordenada

Relação de itens em que a sequencia importa.

Exemplos:

  1. Andressa
  2. Bruno
  3. Carta

Listas aninhadas

podemos misturas listas de diferentes tipos e tambem colocar lista dentro de outras listas.

Exemplos:

  • Doces
    • Brigadeiro
    • Torta de Limão
    • Paçoca
  • Salgados
    • Esfiha
      1. Esfiha de Carne
      2. Esfiha de Frango
    • Pizza
      1. Pizza de Calabresa
      2. Pizza de Portuguesa

Lista de Definição

Lista usada para estruturas de itenns baseados em termo e descrição. Útil para conteudos do tipo: glossários, FAQs, ou apresentação de dados com algum conceito etc.

Exemplos:

Processador
É a unidade de processamento computacional.
Peça essencial em qualquer equipamento digital.
Memória RAM
Espaço de armazenamento volátil.
